Solution: Test with manual token first, then fix Optix configuration.

Step 1: Test API with Manual Token

  • Ensure the URL is set to: http://localhost:8000/test-direct-api.html
  • Verify the canvas is properly configured for token passing
  • Contact Optix support if needed
